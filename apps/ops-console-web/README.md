# Ops Console Web

Next.js operator surface for dispatch, complaint, callcenter, and reporting workflows.

Implemented / materially wired routes include:

- `/dashboard`
- `/dispatch`
- `/callcenter`
- `/complaints`
- `/reports`
- `/incidents`

This app is not a placeholder anymore; it is the active Phase 1 operator console.
The one surface that remains consciously deferred is a dedicated concierge / call-point portal.
That future gate is tracked in `support/sidecars/MSC-P1-001/MSC-P1-001-SURFACE-DECISION-PACKET.md`.
