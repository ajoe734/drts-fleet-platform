DRTS · Design surfaces · v0.4 draft

多功能小客車計程車服務系統 — 完整介面提案

乘車派遣、平台治理、租戶自助、駕駛工作 — 涵蓋四個主要使用者面與一個合作夥伴漏斗。共 50 + 高保真畫面、整合命名與標籤系統、客運法遵與結算邏輯內建在介面語意中。每個入口為獨立 design canvas，可雙擊任一畫面進入聚焦模式比對。
