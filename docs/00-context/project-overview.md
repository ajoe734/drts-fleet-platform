# Project Overview

`drts-fleet-platform` is a bootstrap repository for a multi-surface transportation operations platform.

Current primary product surfaces:

1. tenant portal web
2. platform admin web
3. ops console web
4. driver mobile app

The API, docs, infra, and scripts layers support those surfaces and are included from day one so the repo can grow without structural rework.

This repository is currently in bootstrap state. It contains engineering scaffolding, local development tooling, CI skeletons, and placeholder application shells only.
